# 前端基础笔记：HTML 与 CSS

> 学习路线：先用 HTML 搭建页面结构，再用 CSS 控制外观与布局。每写完一个示例，都在浏览器中打开检查效果。

## 一、开发准备

### 1. 第一个 HTML 页面

HTML 文件扩展名为 `.html`。使用 VS Code 新建 `index.html`，输入 `!` 后按 Enter 可快速生成基础结构。

```html
<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>第一个网页</title>
  </head>
  <body>
    欢迎来到我的第一个网站
  </body>
</html>
```

# HTML

## 二、页面结构与常用标签

### 1. 文档结构

- `<html>`：整个 HTML 文档的根元素。
- `<head>`：页面的配置信息，不直接显示在页面中。
- `<body>`：页面中用户能看到的内容。
- `<title>`：浏览器标签页标题。
- `<meta charset="UTF-8">`：声明页面使用 UTF-8 编码，避免中文乱码。
- `<meta name="viewport">`：让页面在移动设备上按合适的宽度显示。

### 2. 标题、段落与换行

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<p>这是一个段落。</p>
<p>这一行<br>在这里换行。</p>
<hr>
```

- 标题从 `<h1>` 到 `<h6>`，层级应按内容结构使用；一个页面通常只有一个主要的 `<h1>`。
- `<p>` 是段落标签，必须写结束标签 `</p>`。
- `<br>` 用于行内换行，`<hr>` 用于表达主题分隔。

### 3. 图片与路径

```html
<img src="images/bird.webp" alt="一只小鸟" width="200" height="150" title="小鸟">
```

- `src`：图片地址。
- `alt`：图片无法加载或读屏软件阅读时的替代文本；装饰性图片可写 `alt=""`。
- `width`、`height`：显示尺寸，通常不写 `px` 也可；`img` 没有 `size` 属性。
- 图片是行内替换元素，常配合 CSS 控制大小和布局。

路径写法：

```text
images/logo.png      同级目录下的 images 文件夹
./logo.png           当前目录（./ 可省略）
../logo.png          上一级目录
https://example.com/logo.png  网络地址
```

> [!tip] 建议
> 项目内优先使用相对路径，不要使用 `C:\`、`D:\` 这类电脑本地绝对路径，否则换电脑后图片会失效。

### 4. 超链接与文本标签

```html
<a href="https://example.com" target="_blank" rel="noopener">访问示例网站</a>
<p>这是 <strong>重要文本</strong>，这是 <em>强调文本</em>。</p>
<span>span 用于包裹少量行内文本。</span>
```

注意属性名是 `href`，不是 `herf`。`target="_blank"` 表示在新标签页打开链接。

![[html常用文本标签.png]]

### 5. 列表

```html
<!-- 有序列表 -->
<ol>
  <li>安装编辑器</li>
  <li>创建 HTML 文件</li>
  <li>在浏览器中预览</li>
</ol>

<!-- 无序列表 -->
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>梨</li>
</ul>
```

- `<ol>` 用于有顺序的内容；`<ul>` 用于无顺序的内容。
- 每一项都使用 `<li>`。
- Emmet：`ul>li*3` 可快速生成 3 个列表项。

### 6. 表格

```html
<table>
  <caption>月度销售</caption>
  <thead>
    <tr><th>品牌</th><th>销量</th></tr>
  </thead>
  <tbody>
    <tr><td>小米</td><td>100</td></tr>
    <tr><td>华为</td><td>120</td></tr>
  </tbody>
</table>
```

- `<tr>`：一行；`<th>`：表头单元格；`<td>`：普通单元格。
- 合并单元格：横向使用 `colspan`，纵向使用 `rowspan`。

```html
<td colspan="2">横向合并两列</td>
<td rowspan="2">纵向合并两行</td>
```

### 7. 表单

```html
<form action="/login" method="post">
  <label>
    用户名：<input type="text" name="username" required>
  </label>
  <label>
    密码：<input type="password" name="password" required>
  </label>
  <button type="submit">登录</button>
</form>
```

- `action`：提交数据的目标地址。
- `method="get"`：参数会出现在 URL 中，适合搜索、筛选等非敏感操作。
- `method="post"`：数据写在请求体中，常用于登录或提交表单；仍需使用 HTTPS 保护敏感数据。
- `name`：提交时字段的名称；`required`：浏览器要求用户填写。
- `input`、`button`、`img` 常被称为行内块级元素：通常不独占一行，且可以设置宽高。

### 8. 块级元素与行内元素

![[html行内元素与块级元素区别.png]]

| 类型 | 特点 | 常见标签 |
| --- | --- | --- |
| 块级元素 | 默认独占一行，可设置宽高 | `div`、`p`、`h1`、`ul`、`table`、`form` |
| 行内元素 | 默认与文字同行，宽高通常由内容决定 | `a`、`span`、`strong`、`em` |
| 行内块级元素 | 默认不换行，但可设置宽高 | `img`、`input`、`button` |

### 9. 语义化标签

```html
<header>页头</header>
<nav>导航</nav>
<main>
  <section>一个主题区块</section>
  <article>可独立阅读的内容</article>
  <aside>侧边补充内容</aside>
</main>
<footer>页脚</footer>
```

语义化标签让页面结构更易读，也有助于无障碍访问和搜索引擎理解内容。

# CSS

## 三、CSS 基础

### 1. CSS 是什么

CSS（Cascading Style Sheets，层叠样式表）负责页面的颜色、字体、间距和布局。基本语法：

```css
选择器 {
  属性: 值;
}
```

![[css-h1颜色与字号.png]]

### 2. 引入 CSS 的三种方式

```html
<!-- 行内样式：只适合临时测试 -->
<p style="color: orange; font-size: 24px;">文字</p>

<!-- 内部样式：写在当前 HTML 的 head 内 -->
<style>
  p { color: limegreen; }
</style>

<!-- 外部样式：项目开发中推荐 -->
<link rel="stylesheet" href="styles.css">
```

实际项目优先使用外部样式表，便于复用和维护。

## 四、选择器与优先级

```css
* { box-sizing: border-box; }      /* 通配选择器 */
p { color: #333; }                 /* 元素选择器 */
.notice { color: #e67e22; }        /* 类选择器 */
#header { height: 64px; }          /* ID 选择器 */
h1, h2, h3 { font-family: sans-serif; } /* 分组选择器 */
```

![[css-span行内样式.png]]

![[css类选择器.png]]

![[css-id选择器.png]]

优先级可粗略记为：行内样式 > ID 选择器 > 类/属性/伪类选择器 > 元素选择器。优先级相同时，后写的规则覆盖先写的规则。

> [!tip] 建议
> 日常写样式优先使用类选择器；不要依赖大量 `!important`，它会让后续维护困难。

## 五、字体与背景

```css

  color: #222;
  font-size: 16px;
  font-weight: 700;
  font-style: italic;
  font-family: "Microsoft YaHei", "PingFang SC", sans-serif;

  background-color: #f5f5f5;
  background-image: url("bg.png");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}
```

常用单位：`px`（像素）、`%`（相对父元素）、`rem`（相对根元素字号）、`vw` / `vh`（相对视口）。CSS 数值如 `font-size: 16px` 必须带合适的单位；只有 `0` 可以省略单位。

## 六、盒子模型

每个元素都可以看作一个盒子，由内容、内边距、边框和外边距组成：

```css
.box {
  box-sizing: border-box;
  width: 300px;
  padding: 20px;
  border: 1px solid #ddd;
  margin: 16px auto;
}
```

- `content`：内容区域。
- `padding`：内容与边框之间的内边距。
- `border`：边框。
- `margin`：元素与外部之间的间距。
- `box-sizing: border-box`：设置的宽高包含 `padding` 和 `border`，推荐作为全局基础样式。

## 七、Flex 弹性布局

Flex 适合一维布局。父元素设置 `display: flex` 后，它的直接子元素成为弹性项目。

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 16px;
}

.item {
  flex: 1;
}
```

| 属性 | 作用 |
| --- | --- |
| `flex-direction` | 主轴方向：`row`、`column` 等 |
| `justify-content` | 沿主轴对齐：如 `center`、`space-between` |
| `align-items` | 沿交叉轴对齐：如 `center`、`stretch` |
| `flex-wrap` | 子项放不下时是否换行 |
| `flex` | 子项如何放大、缩小及设置基础尺寸 |

> [!note] 记忆方法
> `justify-content` 管主轴，`align-items` 管交叉轴。主轴方向由 `flex-direction` 决定。

## 八、文档流与浮动

### 1. 文档流

默认情况下，块级元素从上到下排列；行内元素从左到右排列。连续的空格、换行和 Tab 在 HTML 中通常会折叠为一个空格。

### 2. 浮动

```css
.avatar {
  float: left;
  margin-right: 12px;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

`float` 最初用于文字环绕图片。现代页面整体布局优先使用 Flex 或 Grid；需要清除浮动时可使用 `clear: both` 或上例的 clearfix。

## 十、易错点速记

- HTML 标签要正确闭合，例如 `</title>`、`</p>`。
- 图片使用 `src`、`alt`、`width`、`height`，不要使用不存在的 `size` 属性。
- 链接属性是 `href`。
- CSS 属性和值之间用冒号 `:`，每条声明以分号 `;` 结束。
- 尺寸通常要写单位，例如 `font-size: 16px`。
- 页面布局优先考虑 Flex；浮动主要用于文字环绕等场景。
